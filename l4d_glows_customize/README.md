## DESCRIPCIÓN:
Gestiona y personaliza el aura de tu personaje con el color de tu preferencia.

PROPIEDADES:
-  Soporte para agregar colores.
- Compatible con Lux's Model Changer(LMC) y Hats.
- Cookies.
- Rainbow Color
- etc...

---

## COMANDOS:
GLOBALES:
```c
 •sm_aura -> Abre el Glow Menu
 •sm_light
 •sm_glow
```

## CONVARS:
El archivo de configuración se genera automáticamente en `cfg/sourcemod/l4d_glows.cfg`.

<details>
<summary>Ver la lista de ConVars y configuración</summary>

```c
// ConVars for plugin "l4d_glows_customize.smx"


// Enable Glows?
// -
// Default: "1"
// Minimum: "0.000000"
// Maximum: "1.000000"
vip_glows_enable "1"

// How long (in seconds) the client will be in 3rd person view after using the glows commands. (0.5 < = off)
// -
// Default: "3.5"
// Minimum: "0.000000"
// Maximum: "360.000000"
vip_glows_thirdperson_time "3.5"

```
</details>
