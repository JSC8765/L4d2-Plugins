# [L4d2] Hunter Multi Jumps [Revisado] (v. 1.1)

Basado en el trabajo de King_OXO: [[L4D2]Hunter Advanced Jump](https://forums.alliedmods.net/showthread.php?p=2744447) - 1.0

---

## DESCRIPCIÓN:
Presiona SPACE como hunter para saltar más veces.

PROPIEDADES:
- Se corrigió el salto manual y la detección del suelo real.
- La trayectoria es rectilínea, el impulso es mucho mayor y dependerá de la mira del jugador.
- Puedes usar esta habilidad sin necesidad de usar el ataque primario.
- etc...

[Demostración](https://youtu.be/69ZMaBPwjRY)

---

## CONVARS:
El archivo de configuración se genera automáticamente en `cfg/sourcemod/l4d2_hunter_multi_jump.cfg`.

<details>
<summary>Ver la lista de ConVars y configuración</summary>

```c
// ConVars for plugin "l4d2_hunter_multi_jump_fixed.smx"


// Hunter Jump Boost
// -
// Default: "300.0"
hunter_jump_boost "300.0"

// Enables Hunter Advanced Jump.
// -
// Default: "1"
hunter_jump_enable "1"

// Hunter Max Jumps
// -
// Default: "1"
hunter_jump_max "1"

// Hunter Advanced Jump Version
// -
// Default: "fix 2.1"
hunter_jump_version "fix 2.1"


```
</details>
